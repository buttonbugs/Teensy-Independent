This repo is composed of the performance testing codes for the melty brain robot [Stuffed Crust](https://github.com/buttonbugs/Stuffed-Crust)

The main codes are in [src](src) folder.

**Board:** Teensy 4.0

**Framework:** Arduino

**Teensy Setup Menu:** [https://www.pjrc.com/teensy/loader_mac.html](https://www.pjrc.com/teensy/loader_mac.html)

**Teensy Loader Download:** [https://www.pjrc.com/teensy/loader.html](https://www.pjrc.com/teensy/loader.html)

**Visual Studio Extension:** [PlatformIO IDE](https://marketplace.visualstudio.com/items?itemName=platformio.platformio-ide)
